# Алгоритм Штрассена
Асимптотика алгоритма Штрассена:

$$T(n) = \Theta(n^{\log_27})  \approx \Theta(n^{2.81})$$

Асимптотика нашего предполагаемого алгоритма:

$$T(n) = a \cdot T(\frac{n}{4}) + \Theta(n^2)$$

Хотим такое $a$, чтобы алгоритм был асимптотически более эффективным в сравнении с алгоритмом Штрассена.

---

Используем мастер-теорему: $k=2, \, a = a, \, b =4.$

Тогда есть три случая: 

- При $\log_27 > \log_4 a > 2 \implies 49 > a > 16 \implies T(n) = \Theta(n^{\log_4a}) < \Theta(n^{\log_27})$

- При $\log_4 a = 2 \implies a=16 \implies T(n) = \Theta(n^2 \cdot \log n) < \Theta(n^{\log_27}) $

- При $\log_4 a < 2 \implies 0 < a < 16 => T(n) = \Theta(n^2) < \Theta(n^{\log_2{7}})$

---

Нам подходят значения $0 < a < 49$.
